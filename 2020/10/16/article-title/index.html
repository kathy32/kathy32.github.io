搭建参考

1. 报错

  • 解决方法：
  • 清除缓存：npm cache clean --force
  • 重新安装：cnpm install

2. 输入hexo init初始化文件夹，接着输入npm install安装必备的组件

  • 注意：需要在Blog文件夹下进行 npm install，然后才可以执行 hexo g 命令
  • hexo g 生成静态文件
  • hexo s 启动服务器，本地访问
